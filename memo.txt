installation des outils necessaire por la compilation android

sudo apt update
sudo apt install -y git zip unzip openjdk-17-jdk python3-pip autoconf libtool pkg-config zlib1g-dev libncurses5-dev libncursesw5-dev libtinfo5 cmake libffi-dev libssl-dev

pour la version recente:
sudo apt install -y git zip unzip openjdk-17-jdk python3-pip autoconf libtool pkg-config zlib1g-dev libncurses-dev cmake libffi-dev libssl-dev libtinfo6 libsqlite3-dev

installation de buildozer:

pip install --user --upgrade buildozer cython


changer la permission dans le buildozer.spec
android.permissions = INTERNET, WRITE_EXTERNAL_STORAGE, READ_EXTERNAL_STORAGE
ajouter des requirements
requirements = python3, kivy, yt-dlp, openssl, certifi

choisir la versio de jdk utiliser:
sudo update-alternatives --config java

run application
